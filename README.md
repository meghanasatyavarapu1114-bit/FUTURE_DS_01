# Business Sales Performance Analytics

## Objective
Analyze business sales data to identify revenue trends, top-selling products, and profitable regions.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

## Key Findings
- Total Sales: 2,297,200.86
- Total Profit: 286,397.02
- West region generated the highest profit
- Central region generated the lowest profit
- Top-selling product: Canon imageCLASS 2200 Advanced Copier

## Insights
- Sales remained mostly stable with periodic spikes.
- West region performed best in profitability.
- Premium products contributed significantly to total sales.
